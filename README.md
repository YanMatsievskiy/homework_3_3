## Тема 3. Операторы цикла
## Задача 3. Star stair
Вам необходимо “нарисовать” в консоли лестницу из символов “*” заданной высоты.

Формат ввода:

Вводится целое положительное число - высота лестницы.

Формат вывода:

Требуется вывести лестницу из звездочек в виде строк.
